Unified API Gateway

The UI is a thin layer over the FastAPI gateway, not a duplicate frontend backend.

Single-worker Queue

Heavy jobs are serialized to protect the machine from concurrent GPU, browser, and download workloads.

Ephemeral Artifacts

Omitting an output directory creates temporary downloadable outputs instead of silently writing to disk.

Spotify Browser Auth

User authorization is API-driven and UI-friendly, rather than being trapped in terminal prompts.

Local-first Processing

The system is designed around a single machine with explicit task boundaries and observable outputs.

Server-rendered UI

FastAPI templates and HTMX keep the frontend small, inspectable, and aligned with the backend contract.
